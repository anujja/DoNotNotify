# Contributing to DoNotNotify

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Open the project in Android Studio
4. Make your changes
5. Test on a device or emulator
6. Submit a pull request

## Pull Request Guidelines

- Keep changes focused and atomic
- Test your changes on a real device if possible (notification listener behavior can differ from emulators)
- Update the README if you add new features
- Follow existing code style (Kotlin, Jetpack Compose)

## Reporting Issues

When reporting bugs, please include:
- Android version
- Device model (if relevant)
- Steps to reproduce
- Expected vs actual behavior

## Feature Requests

Open an issue describing the feature and why it would be useful. Discussion before implementation helps avoid duplicate work.
